# 首页设计文档

## 概述

登录成功后跳转 main.html，默认展示首页。首页是一个不可关闭的 Tab，永远位于 Tab 栏最左端。

## 整体布局

```
+------------------------------------------------------------------+
|                        顶部导航栏 (已有)                           |
+------------------------------------------------------------------+
| Tab 栏 | 首页（不可关闭） | 其他 Tab...                          |
+--------+--------------------------------------------------------+
|        |                                                        |
|  30%   |                         70%                             |
|        |                                                        |
| 用户信息  |                                                      |
| (顶部)   |                      代办事项                         |
|        |                      (CRUD)                            |
| 日历    |                                                      |
| (中间)  |                                                      |
|        |                                                        |
| 公告    |                                                        |
| (底部)  |                                                        |
|        |                                                        |
+--------+--------------------------------------------------------+
```

- **左侧（30%）**：用户信息、日历、公告，垂直堆叠
- **右侧（70%）**：代办事项列表

## 功能模块

### 1. Tab 规则

- 首页 Tab 不可关闭（无 × 按钮）
- 首页永远位于 Tab 栏最左端
- 其他 Tab 可正常打开/关闭

### 2. 用户基本信息（左侧顶部）

展示内容：
- **姓名**：从 localStorage 获取
- **头像**：默认占位头像（无真实图片则显示姓名首字母）
- **手机号**：脱敏显示（例：138****1234）

### 3. 日历（左侧中间）

- 只读展示当前月份
- 显示今日日期（高亮）
- 不可交互（后续扩展）

### 4. 公告（左侧底部）

- 静态列表展示（标题列表）
- 点击无反应（后续扩展详情）

### 5. 代办事项（右侧）

**数据结构：**
```javascript
{
  id: string,        // 唯一标识 (timestamp)
  title: string,     // 待办标题
  content: string,   // 待办内容
  completed: boolean // 是否完成
}
```

**CRUD 操作：**
| 操作 | 功能 |
|------|------|
| 新增 | 点击「新增」按钮，弹出表单填写标题和内容 |
| 查看 | 列表展示所有待办事项 |
| 编辑 | 点击待办项进入编辑模式 |
| 删除 | 滑动或点击删除按钮移除 |
| 完成 | 点击复选框标记完成 |

**数据存储：** localStorage，key 为 `todos`

## 样式风格

- **整体风格**：简洁商务
- **背景色**：白色 `#ffffff`
- **区块分隔**：浅灰边框 `#e8e8e8`
- **主色调**：`#1890ff`（与 Ant Design 一致）
- **阴影**：轻微阴影 `0 2px 8px rgba(0,0,0,0.05)`

## 文件结构

| 文件 | 说明 |
|------|------|
| `html/main.html` | 无需修改，首页通过 TabManager 动态渲染 |
| `assets/script/home.js` | 首页模块，渲染左侧三区块 + 右侧代办 |
| `assets/css/home.css` | 首页样式 |

## 技术实现

1. **Tab 注册**：main.html 加载后注册 `home` Tab，renderFn 加载 `home.js`
2. **模块加载**：复用现有 `loadModule` 机制（约定路径 `assets/script/home.js`）
3. **首页特殊性**：TabManager 增加配置项标记首页不可关闭
