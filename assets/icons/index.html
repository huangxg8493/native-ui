图标库使用说明

1. 引入 SVG Sprite 文件

在页面 HTML 中添加以下代码，加载图标 Sprite：

<script>
  async function loadIconSprites() {
    const files = [
      'assets/icons/ant-design-icons.svg',
      'assets/icons/element-plus-icons.svg'
    ];
    for (const url of files) {
      const resp = await fetch(url);
      const text = await resp.text();
      const div = document.createElement('div');
      div.style.display = 'none';
      div.innerHTML = text;
      document.body.appendChild(div);
    }
  }
  loadIconSprites();
</script>

2. 使用图标

在页面任意位置使用 <use> 引用图标：

<!-- Ant Design 图标 -->
<svg width="16" height="16"><use href="#ant-user"></use></svg>
<svg width="24" height="24"><use href="#ant-setting"></use></svg>

<!-- Element Plus 图标 -->
<svg width="16" height="16"><use href="#el-user"></use></svg>
<svg width="24" height="24"><use href="#el-setting"></use></svg>

3. CSS 辅助类（可选）

引入 icons.css 后可以使用类名：

<link rel="stylesheet" href="assets/icons/icons.css">

<span class="icon anticon">
  <svg><use href="#ant-user"></use></svg>
</span>
